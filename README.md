# 🍜 Danny's Diner SQL Case Study

## 📌 About the Project
This repository contains SQL scripts for the **Danny's Diner case study**, created in MySQL Workbench.  
It is a practice project designed to strengthen SQL skills through **data modeling, queries, and analysis**.

---

## 🎯 Purpose
The goal of this project is to practice SQL fundamentals such as:
- Creating databases and tables  
- Inserting and managing sample data  
- Writing queries for data exploration  
- Performing aggregations and answering business questions  

---

## 📂 Files
- `Danny's Dinner Exercises.sql` → Creates the database (`dannys_diner`), tables (`sales`, `menu`, `members`), and inserts sample data  
- `Danny's Dinner Project.sql` → Contains SQL queries and analysis performed on the dataset  

---

## ❓ Case Study Questions
The following business questions were answered using SQL:

1. What is the total amount each customer spent at the restaurant?  
2. How many days has each customer visited the restaurant?  
3. What was the first item purchased by each customer?  
4. What is the most purchased item on the menu?  
5. Which item was the most popular for each customer?  
6. Which item was purchased first after becoming a member?  
7. Which item was purchased just before becoming a member?  
8. What is the total items and amount spent before membership?  
9. How many loyalty points does each customer have?  
10. How many points do customers A and B have at the end of January?  

---

## 🚀 How to Run
1. Open **MySQL Workbench**  
2. Run `Danny's Dinner Exercises.sql` to create the database and tables  
3. Run `Danny's Dinner Project.sql` to execute the queries and view the analysis
